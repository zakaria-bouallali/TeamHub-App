# TeamHub-App
This project is a full-stack web application designed to manage football teams and their information in a simple and interactive way. It provides users with a clean interface to view, add, update, and delete team records. The application combines a Node.js and Express backend with a React frontend, making it a great example of a modern web app built with separate client and server layers.

The backend exposes REST API endpoints for team data, while the frontend communicates with those endpoints to display the information dynamically. The project also includes Docker support, allowing the whole application to be run easily with containers, along with MongoDB and Redis services for database and caching support.

This app is ideal for learning and demonstrating:

Full-stack development with React and Node.js
REST API creation and consumption
Docker-based application deployment
Database integration with MongoDB
Basic CRUD operations

In short, this project is a practical and beginner-friendly example of a containerized team management application that can be expanded into a larger production-ready system.
